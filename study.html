介绍

是一套用于构建用户界面的渐进式框架,与其他大型框架不同的是，Vue被设计为可以自底向上逐层应用；核心只关注视图层，容易上手 并且便于与第三方库项目进行整合，当与现代化的工具链以及各种支持类库结合使用时，Vue能够为复杂的单页应用提供驱动
以上来自Vue官网--vue的介绍

为什么用vue

首先是因为公司开发都要用框架，不学习使用框架，工作都找不到！
随着vue的学习与应用深入，理解到框架的好处
1.组件化，使我们的工程容易维护，便于后期拓展
2.代码解耦，我们最早使用jQuery，代码相互之间的依赖关系很严重，现在的框架，无论是MVC等，都能帮我们进行分层，更易读写
3.生态环境，主流框架现在都有主流的解决方案
而在vue中，具体的优势是：
1.免除了频繁操作DOM元素，使数据的更改更简单，容易维护利于开发效率
2.响应式系统，监听数据改变而改变页面内容
3.前后端分离，单页面应用用户体验好

一些坑

1.学到vue的过渡动画，想用豆瓣的API做一个电影展示页面。对table用<transition-group>包裹，发现没有作用; 就去官网查，发现是解析DOM模板时的问题
有些 HTML 元素，诸如 <ul>、<ol>、<table> 和 <select>，对于哪些元素可以出现在其内部是有严格限制的。而有些元素，诸如 <li>、<tr> 和 <option>，只能出现在其它某些特定的元素内部。 这会导致我们使用这些有约束条件的元素时遇到一些问题。例如：
我对table内部使用<transition-group>，会被当做无效内容提升到内部，从而渲染错误; 为解决这个问题，可以使用is绑定
<tbody is="transition-group" mode="out-in"> </tbody>
这样，tbody就会被当作一个<transition-group>进行渲染了;具体戳->>解析 DOM 模板时的注意事项

2.使用插值表达式的问题
使用插值表达式{{}}在网速慢时，会出现在页面加载还是{{msg}}这样的形式，用户体验不好。可以对相应元素使用v-cloak,解决闪烁问题
使用v-text默认是没有闪烁问题的，但是他引用的数据会将元素中的原本内容覆盖，而插值表达式不会。

3.axios请求数据，图片403问题
1.images。weserv.nl方案
getImage(url){
// 把现在的图片连接传进来，返回一个不受限制的路径
if(url !== undefined){
return url.replace(/^(http)[s]*(\:\/\/)/,'https://images.weserv.nl/?url=');
}
}
2.no-referrer方案
页头增加
<meta name="referrer" content="no-referrer" />
img方面
<img referrer="no-referrer|origin|unsafe-url" src="{{item.src}}"/>
如a标签等等都可使用

4.vue中使用swiper插件遇到的问题
vue中使用 由于swiper是通过ajax异步加载拿到图片 new swiper（）会先执行，等ajax数据获取之后，dom重新渲染，但此时swiper中并不会有ajax请求的新数据 所以我们需要在 swiper实例中设置属性 observer：true 监听swiper实例子元素的更新而初始化swiper

介绍

简称js，由网景公司布兰登 艾奇花十天时间设计完成。原名LiveScript，后来与sun公司(就是发明Java的那个)合作 改名为JavaScript。本质是一种运行在客户端的脚本语言， 主要做界面的交互效果。
js代码一般写在三个位置，
1.html文件中的script标签中；
2.html元素中；
3.外部js文件引入。 初学常用三种输入输出语句--1.prompt()； 2.prompt()； 3.console.log()

变量

js的交互效果其实就是操作界面的数据，而提到数据就需要我们的变量。变量通俗来讲就是我们存放东西的盒子,可以临时存储数据，通过变量名对数据调用，甚至能够修改数据。
在js中我们通过var来声明变量。
如：var num = 10;这里是声明了变量num并给它赋值为10，整个表达式就是变量的初始化。
js变量的命名规范：
1.变量由 字母、数字、下划线_、美元符$组成；
2.不能以数字开头；
3.不能是关键字如：var for...；
4。见名知意，取名应该有意义；
5.遵循驼峰命名法，第一个单词首字母小写， 后面单词首字母大写；
6严格区分大小写。

数据类型

js变量基本数据类型：
1.数字类型number；
2.字符串类型string；
3.布尔类型Boolean；
4.对象类型object；
5.空类型null；
6未定义underfind。
获取数据类型的方法：typeof()，
数据类型转换：
其他类型转为number 1.parseInt()、2.parseFloat()、3.Number()；
其他类型转为->string 1.String()、2.toString()、3.+' '(空字符串)；

运算符

算术运算符：+、-、*、/、%(取余或取模) 比较运算符：> < >= <= == === != 注：一个=是赋值符号、二个=是等号 只比较数据的值、三个=是全等号 比较值和数据类型。逻辑运算符：&&与、||或、!非。

流程控制

即控制代码按照怎样的结构顺序执行以达到不同的结果。1.顺序结构：程序中大多按从上往下的顺序执行。2.分支结构：根据条件判断执行什么语句， if(条件表达式1){执行语句1}else if(条件表达式2)......{执行语句2}else{执行语句} 如果条件表达式1为真则执行语句1；如果条件表达式为真则执行语句2； 有几个条件就写几个else if；如果以上条件都不满足，则执行else后面的语句，这是一个多选一的过程。 3循环语句：for循环->for(初始化变量;条件表达式;操作表达式){循环体}；
