javaScript
  1. 介绍
    JavaScript：是一种直译式脚本语言，是一种动态的、弱类型、基于原型的语言。用来给页面增加动态功能。
    组成部分：ECMAscript(基本语法)、DOM(文档对象模型)、BOM（浏览器对象模型）
    能做什么？交互效果、表单校验。
    写在哪？<script type="text/javascript">//js代码</script> js代码可以放在网页任何地方 ，建议放在<head>中；
    也可以从外部引入js文件--<script type="text/javascript" src="demo.js"></script>
    注意：一个script标签只能做唯一的事，不能既引入又在里面写js代码
  2. 常用函数
    alert()--弹出提示框，括号内弹出的是提示信息
    document.write()--向页面打印内容
    prompt()--显示可提示用户进行输入的对话框。
  3. 变量的定义
    var a;//声明一个变量a(创造一个容器)
    var b = 100;//声明一个变量b,并且赋初始值为100
    var c ; c = 100//声明一个变量c,并且赋初始值为100
  4. js的注释
    单行注释 //注释的内容
    多行注释 /*注释的内容*/
  5. 双引号的用法
    双引号又要用双引号，将里面的双引号改成单引号，或者将外层的双引号改成单引号
  6. JS的数据类型
    typeof()--检测数据的类型
    number 数值类型 如：123，-123，2.3
    string 字符串类型 比如"虞美人"
